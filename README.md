# Customer Segmentation ML Project
The purpose of this project is to develop a multi-class classification model that helps the company to perform customer segmentation on their new customers.

Library versions:
- Python = 3.11.4
- numpy = 1.24.4
- matplotlib = 3.7.2
- sklearn = 1.3.0
- pandas = 2.0.3
- xgboost = 1.7.6
- shap = 0.42.1
- seaborn = 0.12.2
